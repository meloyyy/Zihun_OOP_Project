# Платформа онлайн-курсів

Курсовий проєкт з дисципліни «Об'єктно-орієнтоване програмування»
**Виконавець:** Зігун Микита Романович, 621п.

---

## Опис проєкту
**Online Courses Desktop** — це десктопний WPF-застосунок для управління навчальним процесом. Програма дозволяє користувачам переглядати каталог доступних курсів, записуватися на них, відстежувати свій прогрес проходження модулів та зберігати стан навчання між сесіями. 

## Технології
* **Мова програмування:** C#
* **Платформа:** .NET 8.0
* **Графічний інтерфейс:** Windows Presentation Foundation (WPF)
* **Збереження даних:** `System.Text.Json`
* **Тестування:** xUnit

## Основний функціонал та ролі

Система підтримує розмежування прав доступу за допомогою двох ролей:

### Роль: Гість
* Перегляд загального каталогу курсів.
* Пошук та фільтрація курсів за ключовими словами.
* Перегляд детальної інформації про курс (категорія, викладач, тривалість, список модулів).
* Обмеження: Неможливо записуватися на курси чи змінювати дані.

### Роль: Зареєстрований користувач
* Запис на обрані курси
* Відмічання пройдених модулів.
* Автоматичний перерахунок відсотка прогресу курсу.
* Експорт та імпорт прогресу у локальний файл `courses_data.json`.

## Структура проєкту
Проєкт розділено на логічні модулі для зручності підтримки та масштабування:
* `Models/` — класи предметної області (`Course`, `CourseModule`, `BaseEntity` тощо).
* `Services/` — сервіси для бізнес-логіки та роботи з файловою системою (`JsonDataService`).
* `Zihun_OOP_Project_2026_Tests/` — окремий проєкт із Unit-тестами.
